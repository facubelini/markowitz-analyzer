¿Qué es la Frontera Eficiente?

La Frontera Eficiente (Markowitz, 1952) es el conjunto de portafolios que ofrecen el mayor retorno posible para cada nivel de riesgo, o el menor riesgo para cada nivel de retorno.

  • 📍 Cada punto = un portafolio con pesos aleatorios distintos
  • 🎨 Color = Ratio de Sharpe (azul bajo → amarillo alto)
  • ★ Estrella verde = Portafolio de Máximo Sharpe (óptimo para la mayoría)
  • ▲ Triángulo naranja = Portafolio de Mínima Varianza (más conservador)

Todo portafolio por debajo y a la derecha de la frontera es ineficiente: existe otro con mejor retorno o menor riesgo con los mismos activos.

Pesos del Portafolio

¿Cómo se calculan los pesos óptimos?

Se generan 5.000 portafolios aleatorios (Monte Carlo) con distintas proporciones de cada activo (pesos que suman 100%). Para cada uno se calculan retorno, volatilidad y Sharpe.

  • Máximo Sharpe: el portafolio con mejor retorno ajustado por riesgo. Es el "óptimo" de Markowitz para un inversor que busca eficiencia.
  • Mínima Varianza: el portafolio más estable. Minimiza las fluctuaciones del capital. Ideal para perfiles conservadores.
  • Pesos Iguales: asigna la misma proporción a cada activo. Sirve como benchmark de comparación.

¿Por qué importa la correlación?

La correlación mide cómo se mueven dos activos en relación entre sí:

  • +1.0 → Se mueven exactamente igual (no diversifica)
  • 0.0 → Movimientos independientes
  • −1.0 → Se mueven en sentido opuesto (máxima diversificación)

La clave de Markowitz: mezclar activos con baja correlación reduce el riesgo total sin sacrificar retorno. Si todos los activos suben y bajan juntos, no hay diversificación real.

ρ(A, B) = Cov(A, B) / (σA × σB)

¿Qué significa cada métrica?
  • Retorno Anual: retorno logarítmico promedio × 252 días.
    μ = promedio(ln(Pₜ/Pₜ₋₁)) × 252
  • Volatilidad Anual: desviación estándar anualizada.
    σ = desv.est.(retornos diarios) × √252
  • Ratio de Sharpe: retorno ajustado por riesgo (tasa libre de riesgo = 5%).
    S = (μ − rf) / σ  ·  S > 1 es considerado bueno
  • Beta: sensibilidad de cada activo respecto al portafolio igual-ponderado de todos los activos cargados. No es el beta de mercado tradicional (vs. S&P 500). Por construcción, la media de todos los betas siempre es 1.0.
    β = Cov(ractivo, rportafolio) / Var(rportafolio)
